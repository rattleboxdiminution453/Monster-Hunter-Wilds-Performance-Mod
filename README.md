## ❓ Simple Troubleshooting

### Game will not start

- Make sure you extracted the download first
- Check that the files are in the right folder
- Remove any file you added most recently
- Restart Windows and try again

### Game runs but feels slow

- Lower shadows first
- Try a lower render scale
- Turn on upscaling if your GPU supports it
- Close other apps before launch

### Image looks blurry

- Raise the render scale a little
- Try a different upscaling mode
- Check your display resolution in game
- Disable any extra sharpening setting if it looks harsh

### Settings do not save

- Make sure the game has write access to its folder
- Run the game once after you copy the files
- Check that Windows did not block the file
- Try again after a restart
